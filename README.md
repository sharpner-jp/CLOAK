# CLOAK - Secret Routing Client

![License](https://img.shields.io/badge/license-MIT-blue)
![Language](https://img.shields.io/badge/language-Node.js-green)

CLOAK is a privacy-focused web client that provides anonymous browsing. It supports **normal mode** and **HTML mode** (directly fetching and rendering HTML).

## Features

- **High-Speed Browsing**: Optimized routing for fast loading.
- **Simple Operation**: Enter a URL or search term and go.
- **Dual Browsing Modes**:
  - **Normal Mode**: Encrypted iframe proxy.
  - **HTML Mode**: Fetch raw HTML and render securely.
- **Tab Management**: Open, close, and reorder tabs.
- **Safe Content Rendering**:
  - Converts `fixed` elements to `absolute`.
  - Limits `z-index` of high-priority elements.
  - Resolves relative URLs for images, links, and styles.
  - Strips `<script>` tags for security.
